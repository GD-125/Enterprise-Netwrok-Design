Configure Layer 3 Switch

enable
configure terminal
hostname CORE_SWITCH
ip routing

Create VLANs

vlan 10
name MANAGEMENT

vlan 20
name HR

vlan 30
name FINANCE

vlan 40
name IT

vlan 50
name WIRELESS

vlan 99
name SERVERS

Configure Routed Interface to ASA

interface GigabitEthernet0/1
no switchport
ip address 172.16.1.2 255.255.255.252
no shutdown

Configure the DHCP Server Port

interface GigabitEthernet0/2
switchport mode access
switchport access vlan 99
no shutdown

Configure SVIs (Gateways)

interface vlan10
 ip address 192.168.10.1 255.255.255.0
 ip helper-address 192.168.99.10
 no shutdown

interface vlan20
 ip address 192.168.20.1 255.255.255.0
 ip helper-address 192.168.99.10
 no shutdown

interface vlan30
 ip address 192.168.30.1 255.255.255.0
 ip helper-address 192.168.99.10
 no shutdown

interface vlan40
 ip address 192.168.40.1 255.255.255.0
 ip helper-address 192.168.99.10
 no shutdown

interface vlan50
 ip address 192.168.50.1 255.255.255.0
 ip helper-address 192.168.99.10
 no shutdown

interface vlan99
 ip address 192.168.99.1 255.255.255.0
 no shutdown

Configure Access Switch Connections

Management Switch

interface FastEthernet0/1
switchport mode access
switchport access vlan 10
no shutdown

HR Switch

interface FastEthernet0/2
switchport mode access
switchport access vlan 20
no shutdown

Finance Switch

interface FastEthernet0/3
switchport mode access
switchport access vlan 30
no shutdown

IT Switch

interface FastEthernet0/4
switchport mode access
switchport access vlan 40
no shutdown

Wireless Router

interface FastEthernet0/5
switchport mode access
switchport access vlan 50
no shutdown

interface vlan50
ip address 192.168.50.1 255.255.255.0
ip helper-address 192.168.99.10
no shutdown

Default Route

ip route 0.0.0.0 0.0.0.0 172.16.1.1
Save
end
write
